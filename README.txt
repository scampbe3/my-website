# My Website

This is a dual-purpose website for English tutoring services and showcasing web development projects.

## Features

- **English Tutoring Services**
  - One-on-One Tutoring
  - Group Sessions
  - Exam Preparation

- **Web Development Portfolio**
  - Interactive Projects
  - Detailed Project Descriptions
  - Responsive Design

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6)
- Responsive Design Principles

## Deployment

Deployed using [Netlify](https://www.netlify.com/) / [GitHub Pages](https://pages.github.com/)
